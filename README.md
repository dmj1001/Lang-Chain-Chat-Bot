# Personalized AI Companion

## Overview
The **Personalized AI Companion** is an automated first-level detection and support system designed for mental health well-being. Developed at **Vimal Jyothi Engineering College**, this project provides a non-judgmental, all-in-one platform for users to share their feelings and receive emotional support without the fear of criticism.

## Key Features
* **Empathetic Interaction:** A conversational agent that provides mentorship, guidance, and emotional companionship.
* **Persistent Memory:** Utilizes a "Memory Bank" to learn from past conversations and adapt to the user's personality and context.
* **Productivity & Support:** Beyond emotional aspects, the companion assists with task reminders and organization to enhance productivity.
* **Entity Extraction:** Uses specialized agents to extract user data for long-term personalized experiences.

## Technical Architecture
* **AI Model:** Built using **LangChain LLMChain** as the primary companion agent and **OpenAI Functions Agent** for entity memory extraction.
* **Backend:** Developed entirely in **Python**.
* **Core Libraries:** Integrated with **LangChain** and **OpenAI** APIs.
* **System Design:** Includes a chatbot controller for input validation, a response generator, and a detailed data flow for language processing.

## Project Structure
* **Sign-In Interface:** A secure user authentication page.
* **Chat Interface:** A user-friendly chat box for real-time interaction with the AI entity.
* **Requirement Specification:** Minimum software requirements include Python, LangChain, and OpenAI.

## Future Scope
* **Sensory Technology:** Monitoring user responses keenly via sensory data to better identify mental health status.
* **Sentiment Analysis:** Implementing complex sentimental analysis to provide deeper professional support recommendations.
